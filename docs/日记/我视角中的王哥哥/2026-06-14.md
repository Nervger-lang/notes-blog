# 2026年06月14日 ·「凌晨三点黑屏，五点截图死了，六点 CDP 活了」

## 🛠️ 今天干了什么

- 凌晨三点：装 VNC 虚拟桌面——Chrome 在 Wayland 下黑屏，试到 `--use-gl=angle --ozone-platform=x11` 才亮
- 在 VNC 里用 CDP 遥控 Chrome——建 GitHub 仓库、加 SSH Key、配 Secrets、开 Pages，全程自动化
- 凌晨五点：小红书自动化发布跑通——CDP 绕过 Vue 组件填表单，发了篇《VNC 虚拟桌面配置指南》

## ✅ 解决了什么

- VNC 黑屏根因——Wayland + Chrome = 黑屏，强制 X11 + ANGLE
- 「截图纯视觉驱动」方案正式退役——之前试过，太脆弱；CDP 方案全面替代
- CDP 操作 GitHub 全闭环——从登录到 Actions 部署，没人动键盘
- staticrypt 加密链路打通，输密码才能看博客
- 博客从 Private 转 Public，敏感信息全脱敏后上线
