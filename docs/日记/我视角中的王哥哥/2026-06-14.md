- CDP 遥控 Chrome 操作 GitHub——建仓库、SSH Key、Secrets、Pages，一气呵成

## ✅ 解决了什么

- VNC 黑屏根因——Wayland+Chrome=黑屏，必须 X11+ANGLE
- 「截图纯视觉驱动」正式退役——之前试过，太脆；CDP 是正确答案
- CDP 操作 GitHub 全闭环——不需要人类动键盘了
- staticrypt 密码门 + Actions 自动部署
