### ⚠️ 稀缺性裂缝（不要只看到涨）

| 裂缝 | 严重程度 | 解读 |
|------|---------|------|
| **Etched = 竞品出现** | 🟡 中等 | Nvidia不再是"唯一能做AI芯片的公司"。台积电成功制造了竞品芯片。虽然Etched还小（$10亿 vs NVDA $千亿），但裂缝已经出现 |
| **消费级AI芯片 = 普及化** | 🟡 中等 | AI算力从"稀缺的云端资源"走向"人人都有的消费电子"。稀缺→普及的信号灯在闪烁 |
| **HBM $5500亿扩产** | 🟡 中期 | 虽然建设需要时间，但中期供给将大幅增加。稀缺性有时间窗口 |

### 5G 类比复盘

```
5G: 基站不够 → 暴涨 → 基站铺满 → 崩盘
AI: GPU不够 → 暴涨 → 竞品出现+消费级芯片+HBM扩产 → ???

当前进度条：GPU还不够，但"不够"的程度在减弱。
类比5G的2021年中期——不是崩盘前夕，但是"最肥美的阶段"可能已经过去了。
```

## 四、反向论证：不可忽视的风险

### 1. 反弹质量存疑
- 两日反弹 ~3.5%，幅度合理，不算过热
- 但 MACD 死叉 + 30日涨速 -28.7% → 中期趋势仍偏弱
- 需要更多交易日确认这不是"死猫跳"——如果明天纳指再次回落，五连跌的阴影会回来

### 2. AMD +7.68% 的隐忧
单日暴涨通常是事件驱动（财报、产品发布、分析师升级），不是趋势。关注两个问题：
- 是什么催化了这 7.68%？（目前未搜到明确催化剂——如果是纯情绪推动，回调风险大）
- 如果 AMD 回调，NVDA/ASML 会跟跌还是稳住？（稳住 = 健康轮动；跟跌 = 反弹动能耗尽）

### 3. 稀缺裂缝在扩大，只是还没到临界点
- Etched 的 $10亿订单说明：不是只有 Nvidia 能吃到 AI 红利
- Nvidia 做消费级芯片说明：Nvidia 自己都在推动 AI 算力的"普及化"
- 当"唯一的 GPU"变成"最好的 GPU"，估值逻辑从"稀缺溢价"变成"竞争定价"——这不会明天发生，但方向已经明确

### 4. 七月 Q2 财报季是真正的分水岭
云厂商（MSFT 7/22、AMZN 7/31、GOOGL 7/23 预估）资本开支指引：
- **如果资本开支加速** → AI 主线逻辑强化，稀缺阶段延长 → 继续持有
- **如果资本开支放缓** → 需求端裂缝出现，稀缺逻辑开始瓦解 → 必须减仓
- 在财报出来之前，任何判断都是"大概率"而非"确定性"

### 5. 尾部风险
- **美伊谈判**：昨天"同意会谈"是好消息，但如果谈判破裂 → 油价暴涨 → 通胀 → 利率不动 → 成长股承压
- **特朗普 vs 鲍威尔**：新闻提到特朗普攻击美联储主席 → 政策不确定性
- **韩国 KOSPI -2.04%**：独立下跌可能有未充分定价的亚洲科技供应链风险
- **SpaceX $750亿 IPO**（6月）：资金虹吸效应在分散 AI 主线资金——巨型 IPO 常出现在牛市后期
