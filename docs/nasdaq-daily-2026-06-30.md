**风险因素**：+2.07% 单日涨幅如果今晚无法接力，可能触发获利回吐。NQ 期货亚洲时段仅 +0.54%（远小于昨日的 +2.43%），动能有所减弱。

### 2. 估值合理吗？

纳指 25,820，PE 约 28-30x（估计）。相比 6/18 高点 26,518 时的估值已经回落。收复 50MA 意味着中期趋势尚未破坏。

**正面因素**：美伊会谈缓解了「油价暴涨→通胀→利率」链条，成长股估值压力减轻。

**风险因素**：7 月 Q2 财报如果云厂资本开支指引下调，当前估值需要重新定价。**当前反弹的基础是「产业界的长期承诺（$5500亿）」+「地缘风险缓解」——两个都是预期，不是业绩兑现。**

### 3. 稀缺逻辑有裂缝吗？

**改善的信号：**
- 🟢 韩国 $5500 亿投资 → 产业界一致判断供不应求将持续

### 4. 市场叙事在转变吗？

**从「AI 到底行不行 + 中东打起来了」→「AI 还很行 + 中东要谈了」。**

一周内叙事 180° 翻转：
- 6/26：ASML 沉默、芯片全绿、美伊互射 → 「主线失血 + 地缘危机」
- 6/29：ASML 高歌、芯片全红、美伊会谈 → 「主线修复 + 地缘缓解」

**这种速度的叙事翻转本身就值得警惕。** 不是「信号」，是「信号的速度」。如果今晚一个利空就能让 ASML 再次翻绿、芯片再次全绿，说明市场在「极度悲观」和「极度乐观」之间剧烈摇摆——这不是稳健的底部，而是情绪驱动的波动。
