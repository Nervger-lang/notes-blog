# 主线+稀缺性 市场分析方法论

> 2026-06-12 | 与 Hermes 对话整理

---

## 一、核心思想

**技术指标只做参考，真正的行情驱动力是"稀缺性"。**

不是卖得越好越有行情——是**越稀缺、越没普及，行情越好**。

类比：5G 在基站不够、芯片不够的时候暴涨。基站铺满、芯片过剩之后暴跌。股票在稀缺阶段见顶，不需要等到普及才跌。

---

## 二、三步分析法

### 1. 找到主线

当前市场涨的根本驱动力是什么？形成完整产业链条：

```
上游（最稀缺）→ 中游 → 下游（应用）
```

例子：AI 主线

```
英伟达 GPU → 台积电 CoWoS → HBM 内存（SK海力士/三星）
    ↓
数据中心光模块 → 云厂商资本开支 → 企业 AI 应用
```

### 2. 判断稀缺性

四个问题：

| 维度 | 问题 | 当前 AI 答案 |
|------|------|------------|
| 供给 | 核心环节有瓶颈吗？ | ✅ CoWoS 排到 2027，HBM 供不应求 |
| 需求 | 需求在加速还是减速？ | ✅ Q1 四大云厂 capex $1,306 亿，创纪录 |
| 替代 | 有替代品吗？ | ⚠️ AMD 追赶、云厂自研芯片、DeepSeek 提效率 |
| 普及 | 终端用起来了吗？ | ❌ 企业 AI 大多还在 PoC，没普及 |

### 3. 定位阶段

| 阶段 | 信号 | 操作 |
|------|------|------|
| 🔵 稀缺初期 | 供给严重不足，需求刚起步 | 重仓 |
| 🟢 稀缺高峰 | 供给仍不足，但市场问 ROI | 持有，警惕 |
| 🟡 稀缺末期 | 替代品出现，效率提升 | 逐步减仓 |
| 🔴 普及期 | 供给追平，利润压缩 | 清仓 |

**关键：稀缺末期 ≠ GPU 烂大街。** 稀缺末期的信号是效率提升（DeepSeek）、竞争对手追上来、市场开始质疑 ROI。这些信号出现时就要警惕，不需要等到真的过剩。

## 三、反向论证（必做）

每次分析完正面逻辑，**强制推演反面**：

1. **涨太快了吗？** — 短期涨幅过大，正常回撤可能远超预期
2. **估值合理吗？** — 盈利增速跟得上股价吗？还是纯拔估值？
3. **稀缺逻辑有裂缝吗？** — 效率突破、替代品、政策变化？
4. **市场叙事在变吗？** — "改变世界" → "证明给我看" 是最危险的转折
5. **尾部风险？** — 地缘政治、单一故障点、黑天鹅

**不顺着用户说，要找逻辑漏洞。** 这是分析质量的生命线。

- 市场叙事从"改变世界"转向"证明给我看"
